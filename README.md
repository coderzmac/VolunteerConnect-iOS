# VolunteerConnect

## 📱 Overview
VolunteerConnect is an iOS application designed to help local nonprofit organizations efficiently manage volunteer scheduling, communication, and event coordination. The app provides a centralized platform for both volunteers and coordinators, reducing reliance on spreadsheets, emails, and manual tracking systems.

---

## 🎯 Problem Statement
Many nonprofit organizations rely on outdated or manual systems to manage volunteers, leading to:
- Scheduling conflicts
- Poor communication
- Volunteer no-shows
- Increased administrative workload

VolunteerConnect addresses these challenges by providing a modern, mobile-first solution.

---

## 🚀 Features
- User authentication (Volunteers & Coordinators)
- View and sign up for volunteer events
- Create and manage events (Coordinator role)
- Shift scheduling and assignment
- Push notifications and reminders
- Attendance tracking

---

## 🧱 Tech Stack
- **Language:** Swift
- **Framework:** SwiftUI
- **Architecture:** MVVM (Model-View-ViewModel)
- **Backend:** Firebase (Authentication + Firestore)

---

## 🧩 Object-Oriented Design
The application is built using object-oriented principles with core classes such as:
- `User`
- `Volunteer`
- `Coordinator`
- `Event`
- `Shift`
